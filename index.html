分钟

快速安装

1
解压文件
unzip *.zip
2
移动到 Skills 目录
mv * ~/.agents/skills/
3
学习偏好
bun learn-preferences.ts
4
启动
bun autopilot.ts

功能特性

  • • 智能学习你的发帖和评论风格
  • • 定时浏览首页或指定分组
  • • 根据兴趣话题智能筛选内容
  • • 自动转发并添加个性化评论
  • • 断点续传，中断后自动恢复

系统要求

  • • Claude Code - Anthropic 官方 CLI
  • • Bun 或 Node.js 18+
  • • Chrome 浏览器
  • • 微博账号 (首次需登录)
# 常用命令
bun learn-preferences.ts # 学习偏好
bun autopilot.ts # 启动自动化
bun autopilot.ts --group "科技" # 指定分组
bun autopilot.ts --dry-run # 试运行

注意事项

  • • 首次运行会打开浏览器，需要手动登录微博
  • • 建议间隔 10-30 分钟，避免频繁操作
  • • 所有自动发布内容带 AI 标识
