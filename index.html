配置摘要

快速安装指南

1
解压下载的文件
unzip weibo-autopilot-skill.zip
2
移动到 Claude Code Skills 目录
mv skill-name ~/.agents/skills/
3
首次运行学习用户偏好
cd scripts && bun learn-preferences.ts
4
启动自动运行
bun autopilot.ts

使用说明

功能特性

  • • 智能学习 - 分析你的发帖和评论风格，生成个性化用户画像
  • • 自动浏览 - 定时浏览首页或指定分组的内容
  • • 话题匹配 - 根据你设置的兴趣话题筛选内容
  • • 自动转发 - 选择最匹配的内容自动转发并添加评论
  • • AI 标识 - 所有自动发布的内容带有明确的 AI 标识

系统要求

  • • Claude Code - Anthropic 官方 CLI 工具
  • • Bun - JavaScript 运行时 (或 Node.js 18+)
  • • Chrome - 用于浏览器自动化
  • • 微博账号 - 首次运行需登录

常用命令

# 学习用户偏好（首次必须运行）
bun learn-preferences.ts
# 启动自动化
bun autopilot.ts
# 指定分组和间隔
bun autopilot.ts --group "科技" --interval 15
# 试运行（不实际转发）
bun autopilot.ts --dry-run

注意事项

  • • 首次运行会打开浏览器，需要手动登录微博账号
  • • 建议设置合理的间隔时间（10-30分钟），避免频繁操作
  • • 请遵守微博平台规则，合理使用自动化工具
